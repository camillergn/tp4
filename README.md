# Projet TP4 - AlgoBox

## Fichiers
- `somme.alg` : fichier principal du projet, permettant de tester le workflow collaboratif avec GitHub.

## Exécution
1. Ouvrir le fichier `somme.alg` avec **AlgoBox**.
2. Lancer l’algorithme depuis AlgoBox.
3. Suivre les instructions affichées à l’écran pour saisir les nombres et visualiser le résultat de la somme.

## GitHub
- Dépôt : [https://github.com/camillergn/tp4](https://github.com/camillergn/tp4)
- Version figée (tag/release) utilisée pour le TP : `v1.0`

## Notes
Ce projet a pour but de **tester le workflow collaboratif** via GitHub : récupération des modifications (`git pull`), enregistrement des changements (`git commit`) et synchronisation avec le dépôt (`git push`).
